我的權限可以做什麼？

?
已啟用的普通帳號可以維護資料庫內容與建立備份；備份管理員可還原與刪除備份；超級管理者另可審核帳號。
尚未登入：目前只能使用排菜、查詢與匯出等公開功能。

未登入訪客

  • 可以使用排菜總覽、查看資料、匯出菜單與庫存/食材/菜色 Excel。
  • 不能修改資料庫、建立/還原/刪除備份，也不能審核帳號。

普通帳號（active user / manager）

  • 可以新增、編輯、刪除食材、菜色、價格、庫存、單位換算與菜色食材清單。
  • 可以建立資料庫備份與補充備份註解，方便資料維護前後留存快照。
  • 不能審核/刪除帳號，不能還原備份，也不能刪除備份檔。

備份管理員（backup_manager）

  • 包含普通帳號所有資料維護能力。
  • 可以還原備份、刪除單一備份或批次刪除備份。
  • 不能審核、拒絕或刪除帳號。

超級管理者（superuser）

  • 包含備份管理員與普通帳號所有能力。
  • 可以審核、拒絕、刪除帳號與指定帳號角色。
